# Online Banking System Implementation

## Project Overview
This project is a web-based implementation of an Online Banking System. The interface simulates core banking functionalities including user dashboard, fund transfer, and bill payments.

**Research Paper:** Design and Implementation of an Online Banking System

## Domain
FinTech / Banking

## Description: 
This project implements the web interface for an Online Banking System as proposed in the research paper, featuring a secure login, user dashboard, fund transfer, and bill payment functionalities.

## Features
* **Login Interface:** Secure entry point for users.
* **Dashboard:** Displays real-time account balance and recent transactions.
* **Fund Transfer:** Form to simulate money transfers to beneficiaries.
* **Bill Payments:** Interface for utility bill settlements.
* **Transaction History:** Tabular view of past credits and debits.
* **Profile:** User details and account information.

## Technologies Used
* HTML5
* CSS3
* JavaScript

## Folder Structure
* `index.html` - Login Page
* `dashboard.html` - Main Dashboard
* `transfer.html` - Fund Transfer
* `bills.html` - Bill Payments
* `history.html` - Transaction History
* `profile.html` - User Profile
* `style.css` - Stylesheet
